  • create(config: List.ICreateOptions): void
    • Defined in labkey/List.ts:49

    Create a new list. A primary key column must be specified with the properties 'keyName' and 'keyType'. If the key is not provided in the domain design's array of fields, it will be automatically added to the domain.

     LABKEY.List.create({
         name: "mylist",
         keyType: "int",
         keyName: "one",
         description: "my first list",
         fields: [{
             name: "one", rangeURI: "int"
             },{
             name: "two", rangeURI: "multiLine", required: true
             },{
             name: "three", rangeURI: "Attachment"
         }]
     });
